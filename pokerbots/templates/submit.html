📖 Quick Start

1. Download the template

Get the starter bot from the organizer. Your zip should look like:

your_team.zip
├── player.py        ← your code goes here
└── commands.json    ← {"build":[],"run":["python3","player.py"]}
2. Implement get_action()

This is the only function you must implement. It's called every time it's your turn.

def get_action(self, game_state, round_state, active):
    legal_actions = round_state.legal_actions()

    # Your cards
    my_cards  = round_state.hands[active]
    board     = round_state.deck[:round_state.street]

    # Chip counts
    my_stack  = round_state.stacks[active]
    my_pip    = round_state.pips[active]
    opp_pip   = round_state.pips[1 - active]

    # Raise bounds
    if RaiseAction in legal_actions:
        min_r, max_r = round_state.raise_bounds()
        return RaiseAction(min_r)

    if CheckAction in legal_actions:
        return CheckAction()
    return CallAction()
3. Test locally
python player.py --port 10000
4. Zip and submit
# Mac / Linux
zip -j my_team.zip player.py commands.json

# Windows (PowerShell)
Compress-Archive player.py,commands.json my_team.zip

Then upload the zip on this page. You can resubmit as many times as you like before the tournament starts.
